HR Data Analysis Project (Excel + SQL)

Overview:
This project analyzes an HR dataset to uncover insights related to attrition, demographics, job satisfaction, and workforce structure. Work was done using Excel (data + dashboard) and SQL.

Project Structure:
HR Data Analysis.sql
HR_Dashboard.xlsx
README.txt

Tools Used:
- Microsoft Excel
- SQL Server / PostgreSQL
- SQL (group by, crosstab, aggregation)

Key SQL Tasks:
- Employee Count
- Attrition Count
- Attrition Rate
- Active Employees
- Average Age
- Attrition by Gender
- Attrition by Department
- Employees by Age Group
- Education Field-wise Attrition
- Gender + Age Band Attrition
- Job Satisfaction Crosstab

Insights:
- Highest attrition occurs in specific departments
- Age groups show different attrition patterns
- Job satisfaction varies across roles
- Education fields have different attrition counts
